
# Jarvis AI Assistant
A Python-based virtual assistant that uses Google Speech Recognition to execute commands.

## 🚀 Features
- **Voice Activation:** Listens for the wake word "Jarvis."
- **Modular Design:** Separate modules for API handling (`client.py`) and media (`musicLibrary.py`).
- **Robust Error Handling:** Manages ambient noise and speech timeouts effectively.

## 🛠️ Tech Stack
- **Language:** Python
- **APIs:** Google Speech Recognition, OpenAI (optional)
- **Libraries:** SpeechRecognition, PyAudio, pyttsx3

## ⚙️ Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the project: `python main.py`
